# Contributing

Thanks for your interest in SimpleSSHTerm!

## Development setup

1. Install dependencies (Qt5, libssh, libsodium, libvterm, pkg-config, cmake).
2. Build:

```bash
cmake -S . -B build
cmake --build build
```

## Code style

- Keep changes small and focused.
- Prefer clear, minimal UI changes.
- Avoid introducing new dependencies without discussion.

## Issues and PRs

- Open an issue for bugs or feature requests.
- For PRs, include a short description and test steps.
