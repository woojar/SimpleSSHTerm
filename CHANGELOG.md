# Changelog

## v0.1.0-beta.1

- Embedded SSH via libssh
- Tabbed sessions with optional connect-in-new-tab per profile
- Encrypted profile storage with passphrase toggle
- SSH config import and key-based auth
- Reconnect last sessions on startup
- Theme editor and Base16 theme import
- ANSI/VT rendering via libvterm (fallback if unavailable)
- Mouse selection copy/paste in terminal
